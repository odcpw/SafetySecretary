## de
Title: Exporte
Audience: Nutzer, die Dokumente teilen oder archivieren
Summary: Exporte erzeugen Office- und PDF-Dateien aus den aktuellen Daten oder Snapshots.
How to:
- Waehle den Workflow und das Ausgabeformat.
- Nutze freigegebene Snapshots, wenn die Ausgabe verbindlich sein soll.
- Pruefe die Datei vor dem Teilen.
See also: hira, toolbox-talk, privacy, self-host
Body:
HIRA, JHA und II erzeugen Berichte und abgeleitete Artefakte. Dateien werden gespeichert, damit handbearbeitete Fassungen nicht still ueberschrieben werden.

## en
Title: Exports
Audience: Users sharing or archiving documents
Summary: Exports create Office and PDF files from current data or approved snapshots.
How to:
- Choose the workflow and output format.
- Use approved snapshots when the output must be stable.
- Review the file before sharing.
See also: hira, toolbox-talk, privacy, self-host
Body:
HIRA, JHA, and II produce reports and derived artifacts. Saved files are retained so hand-tuned versions are not silently overwritten by regeneration.
